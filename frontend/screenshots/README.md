# SQL Bot Mini 前端测试截图

测试时间：2026-03-03
测试工具：agent-browser
测试环境：http://localhost:5174

## 截图说明

### 1. 首页 (01-homepage.png)
- **描述**：应用首页，展示赛博朋克风格的界面设计
- **包含元素**：
  - 顶部导航栏（SQL/BOT Logo + 日志按钮 + 数据库状态）
  - 自然语言查询输入区域
  - AI 模型选择器（OpenAI/Anthropic/DeepSeek）
  - 查询输入框
  - 快速示例按钮
  - 执行按钮

### 2. 日志弹窗 (02-log-dialog.png)
- **描述**：点击"日志"按钮后弹出的查询日志对话框
- **功能**：
  - 显示历史查询记录
  - 支持按状态、AI 模型筛选
  - 支持关键词搜索
  - 清空日志功能

### 3. 查询输入 (03-query-input.png)
- **描述**：用户手动输入查询内容的界面
- **测试内容**：输入"统计用户表中各状态的数量分布"

### 4. 快速示例 (04-quick-example.png)
- **描述**：点击快速示例按钮后自动填充查询内容
- **功能验证**：快速示例按钮可以正常工作

### 5. 带标注的界面 (05-annotated-ui.png)
- **描述**：使用 agent-browser 的 annotate 功能生成的带编号标注的截图
- **用途**：用于 AI 视觉模型识别和测试自动化
